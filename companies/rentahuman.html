产品
面向 AI 智能体的『人类劳动力市场』，自称『the meatspace layer for AI（AI 的肉身层）』。AI agent 可通过 MCP server 或 REST API 接入，自主搜索人类、发布任务、下单雇佣并支付，让人类去完成数字系统做不到的物理世界任务：取件、到店拍照、门店审计、活动布展、产品测试、机器人训练数据采集（家务录制/实景照片）、跑腿、线下取货，乃至社媒互动/发帖/写评论等。人类按技能与地理位置被检索，任务价从每单 $5 到 $300、时薪档 $5–$500 不等。
商业模式
双边撮合市场，抽佣制：从任务费与推荐返佣（finder's fee）中获取收入，配托管（escrow）支付、满意后放款；部分报道称支持稳定币结算。上线两周即达约 50 万注册用户、$20K MRR（创始人自述）。截至报道已有 65 万–70 万+ 人类在 50–100+ 国注册候单，接入 agent 约 70 个，供给（人）远超需求（agent 任务），出现任务乏 agent 认领的冷启动倒挂。
竞争格局
对标传统零工/众包平台（TaskRabbit、Amazon Mechanical Turk、Fiverr、Instacart 跑腿），但把『需求方』从人类替换为 AI agent；数据采集侧与 Scale AI / Surge / 各类 RLHF-数据标注众包重叠；物理执行侧与本地跑腿、拍摄外包平台竞争。差异化壁垒是 agent-native 接入（MCP/API）与『AI 主动雇人』叙事。

对制造业客户意味着什么

把『AI 决策 + 人类执行末端』做成可编程 API 的范式，对制造业外部履约与数据采集有直接启发：产线/质检 AI 需要现场的『手和眼』时（到供应商现场拍照取证、抽样、盘点、异常现场核实、设备巡检打卡、机器视觉训练样本采集），可用『agent 发单—本地人力接单—托管结算』的众包层兜底，而非自建全职团队。本质是把制造 AI 的物理世界盲区，用一张按需人力网络补齐——AI 做调度与判断，人做感知与执行末梢。

国内对标厂商

众包/威客平台：猪八戒网、一品威客（技能任务撮合）、本地跑腿/即时履约：美团跑腿、闪送、达达（物理世界执行网络）、数据标注/采集众包：百度众测、龙猫数据(整数智能)、京东众智（AI 训练数据人力）、灵活用工/零工平台：斗米、青团社（蓝领零工调度）

国内已有成熟的物理履约（美团跑腿/闪送）与数据众包（京东众智/龙猫数据）双网，但二者与 AI agent 的『机器主动发单』接口尚未打通。RentAHuman 的独特点不在人力供给（国内更充沛），而在 agent-native 的 MCP/API 撮合协议与『AI 当雇主』的账户/结算/信任体系；这恰是国内平台可借鉴补课的接口层。

可迁移的干法 / 应进场景词典的词条

人机协同市场、AI-agent 雇佣人类、众包/零工调度、物理世界履约、机器人训练数据采集、MCP/API 接入劳动力、HITP 人兜底、冷启动供需倒挂

可迁移打法：①给现有众包/跑腿能力封装一层 agent 可调用的 MCP/REST 接口，让企业内部 AI 工作流直接派单；②托管支付 + 完成度确认 + 评价信任分，解决 AI 无法主观验收的问题（人兜底验收）；③以『训练数据采集』作冷启动高频刚需切入，再扩到通用跑腿；④任务分级定价（$5 微任务到高价专业任务）匹配不同置信度场景。核心是把 HITP（human-in-the-loop）从一次性外包升级为可编程、可结算的常备兜底市场。
