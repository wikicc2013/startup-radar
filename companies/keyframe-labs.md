## 转型有术判断
1. **对制造业客户意味着什么**：对以制造业为主体的企业而言，Keyframe 代表「营销·销售·客服」环节从文本/语音客服升级为「可对话的真人视频界面」这一步。制造企业产品复杂、决策链长，面客场景（渠道商培训、售前技术答疑、复杂设备的远程指导、经销商与一线销售的合规话术演练）过去依赖真人且难规模化。这类实时视频数字人可低成本承载：① 产品与合规知识的角色扮演式销售/客服培训（L&D）；② 面向海外经销商/客户的多语言售前视频接待与演示，比静态聊天机器人转化更高；③ 售后设备操作/安装的可视化交互指导。核心价值是把「专家的脸和话术」规模化复制到每一次面客触点，且 0.06 美元/分钟的边际成本使其在高频、长尾场景可行。
2. **国内对标厂商**：硅基智能（硅语/数字人直播与实时交互，最贴近的实时对话数字人对标）、魔珐科技 Mofa（超写实数字人+实时交互，偏 B 端定制）、商汤「如影」SenseAvatar（可控数字人生成，实时能力在补齐）、腾讯智影 / 百度曦灵 / 阿里通义（大厂平台化数字人，覆盖生成+直播+客服接口）
   > 国内实时对话数字人赛道已相当拥挤：硅基智能、魔珐主打商用落地与直播带货，大厂（腾讯智影、百度曦灵、阿里、商汤如影）以平台化 API 供给。相比之下 Keyframe 的差异点是「面向海外开发者的极简 embed 接入 + 低延迟实时通话 + 激进低价（0.06 美元/分钟）+ 强调基础模型自研」。国内玩家在成本与工程化上并不弱，Keyframe 的护城河更多押注在真实感（跨过恐怖谷）与实时并发的模型质量，而非单纯功能覆盖——这也意味着对国内团队而言，真正的门槛是实时视频合成的模型质量与延迟工程，而非产品形态。
3. **可迁移的干法 / 应进场景词典的词条**：实时对话数字人、视频客服/售前接待、销售话术角色扮演培训、招聘面试模拟、远程可视化指导、embed 低代码接入
   > 可迁移干法：把「专家/销售/客服的脸+话术」封装成可 API 调用的实时视频人格，嵌到官网、培训系统、经销商门户里。对制造业最现实的三个切入点是——① 经销商/一线销售的合规话术与产品知识角色扮演训练（用数字人当陪练，可评分）；② 面向海外买家的 7×24 多语言视频售前接待与产品演示；③ 复杂设备安装/售后的可视化交互问答。落地关键是把数字人接到企业自有知识库/Agent 上（Keyframe 只做「脸和实时通话」这一层，业务逻辑仍需企业侧 LLM 供给）。
4. **风险与存疑点**：① 极早期、团队仅 2 人、无公开融资与客户，交付与稳定性未经规模验证，采购需谨慎；② 赛道极度拥挤且大厂（HeyGen/Tavus 海外、腾讯/百度/阿里/商汤国内）资源碾压，0.06 美元/分钟的低价策略可持续性与毛利存疑，易陷价格战；③ 技术护城河集中在「实时视频合成模型质量」这一单点，一旦上游开源/大厂模型追平实时+真实感，产品形态易被替代；此外真人数字人涉及肖像权、深度伪造合规与滥用（诈骗/仿冒）风险，出海与企业采购的合规审查会是硬门槛。
