## 转型有术判断
1. **对制造业客户意味着什么**：对制造企业的浏览器自动化而言,StableBrowse 代表『给 Agent 持久记忆的浏览器层』。制造企业让 Agent 跨网站/门户(报关、供应商、政务)操作时,Agent 每次从零摸索、易失败、token 贵。StableBrowse 把网站转执行图、复用记忆。启示:制造企业的跨门户 Agent 操作(报关/供应商门户/政务)可迁移『浏览器层(网站转执行图+持久记忆)』提升 Agent 操作可靠性、降 token,与 kaizen 同指向浏览器自动化基础设施。
2. **国内对标厂商**：各类浏览器自动化、Browserbase 类、RPA 浏览器
   > 国内浏览器自动化多为 RPA。StableBrowse 差异在于网站转执行图+持久记忆+降 70-80% token。对国内制造企业跨门户 Agent 操作可迁移。
3. **可迁移的干法 / 应进场景词典的词条**：Agent浏览器层、网站转执行图、持久记忆、降token、高成功率、AI基础设施
   > 可迁移干法:跨门户 Agent 操作(报关/供应商/政务)用『浏览器层(网站转执行图+持久记忆)』提可靠、降 token。对制造企业适用。
4. **风险与存疑点**：①Browserbase 等在场;②网站变更致执行图失效;③早期;④反爬对抗。
